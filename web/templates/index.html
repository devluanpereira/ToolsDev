Perguntas Frequentes

O que é a ToolsDev?

A ToolsDev é uma plataforma que oferece ferramentas online gratuitas e de código aberto para auxiliar desenvolvedores e empresas em diversas tarefas.

As ferramentas são gratuitas?

Sim, todas as ferramentas disponíveis na ToolsDev são completamente gratuitas para usar.

Posso contribuir com o projeto?

Com certeza! A ToolsDev é um projeto OpenSource e ficaremos muito felizes com a sua colaboração. Veja nossa página no GitHub para mais informações.

Como posso entrar em contato com o suporte?

Atualmente, oferecemos suporte através da nossa página no GitHub e da nossa comunidade online. Em breve, teremos mais canais de comunicação.
